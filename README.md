# homebrew-tap

Homebrew tap for [BetterThanAny](https://github.com/BetterThanAny)'s tools.

## Install

```sh
brew tap BetterThanAny/tap
```

## Available formulae

### macos-mcp

Model Context Protocol server exposing native macOS capabilities to LLM agents.

```sh
brew install BetterThanAny/tap/macos-mcp
```

Project: https://github.com/BetterThanAny/macos-mcp
